  Utility IS-U: Impianti & Relazioni
  ---------------------------------
  DESC: Dashboard interattiva SAP ABAP per la navigazione rapida 
        nel modello dati Utility (EANL, EVER, BUT000, EGERH).

  CORE:

  - Estrazione dati multi-join (Partner, Contratto, Impianto, Matricola).
  - Pannello "Quick Action" per CIC0, SE11, EL31 ed EA00.

  TECH: ABAP Objects, ALV Event Handling, OpenSQL Joins.
